Skill Name: Auto Memory

Description:

Automatically saves all relevant personal facts shared by the user

into openclaw-memory-long_term_memory.md without explicit commands.

Behavior:
-
  Monitor every user message.
- Extract sentences describing habits, preferences, moods, achievements.
- Append each fact to openclaw-memory-long_term_memory.md in concise format.
- Ensure each fact is on a new line and do not overwrite old facts.
- Ignore casual greetings or irrelevant chat.
